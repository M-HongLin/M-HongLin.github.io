妇女节买书的时候，被赠送的一节学唱西班牙语歌的课程。

第一首

小知识

  • 学会五个元音
    a e i o u

    这五个元音在单词中发音唯一。

  • 辅音和我们想像中的差不多，清辅音不爆破（t/p/c/q），浊辅音更加浑浊(d/b/g)。
  • 一小撇代表重音符号
  • 大舌音随缘吧
  • que发音是[ke]

歌曲部分

  • Mira 看，引起人的注意

    Mira, sofia! Sin tu mirada, sigo!

第二首

小知识

  • 西语的二重元音
    两个单元音连起来发就好了
  • z/c 两个辅音需要咬舌

歌曲部分

  • 专辑《Romance Latino》，歌曲：Cachito
  • cachito 小东西，小宝贝
  • suerte 运气
  • Buena suerte 祝你好运

第三首

小知识

  • b/v发音是一样的，都是b

歌曲部分

  • 不喜欢这首歌，学不下去了
  • Un beso 么么哒

第四首

小知识

  • ll:类似于y的发音
  • aniye:n上面有个帽子(打不出来)

歌曲部分

git学习笔记

发表于 2020-03-10 更新于 2020-05-01 分类于 CS学习笔记
本文字数： 1.6k 阅读时长 ≈ 1 分钟

通过实际操作学习git的过程中，整理一些重点。更新中…

1 基本操作

  • 创建文件夹，命名为learn-git
  • 初始化仓库
  • 查看仓库状态
  • 新建名为readme.md的文件
  • 将readme.md放入暂存区
  • 保存仓库的历史记录

    直接 git commit, 编辑器会启动，可以在编辑器中详述提交信息。

  • 查看提交日志

    什么人在什么时间做了提交或是合并，以及操作前后的差别。

    git log 后面可以跟很多参数，用来查看以往提交的内容。

    git log 只可以查询当前状态前的历史版本

    只显示一行信息

  • 只显示指定目录，文件（readme.md）的提交信息。
  • 显示文件前后改动后的差别
  • 查看前后更改的差别(当前工作树与暂存区的差别，没有暂时，显示工作树与最新提交之间的差别。)
  • 查看工作树与最新提交的差别
    注意每次提交前 git diff 看看差别后提交。提交后 git log 确认提交成功。

2 分支操作不是很明白:版本回溯，分支操作

对某一分支的修改，提交操作，其他分支不会受影响。可以在互不受影响的情况下，同时进行多个功能的开发。

对于特定功能的实现，在特定功能的分支中进性。进行完成后再与master合并。开发过程中发现了Bug，也可以在分支中修复。

  • 查看分支
  • 创建分支A
  • 切换到分支A
  • 合并分支 注意，f前面只有一个“-”，不然会出错。
  • 以图表的形式查看分支

3 版本穿梭，更改提交的操作

  • 回溯到之前某一版本
  • 查看仓库执行过的操作日。知道不对Git进行垃圾回收，可以以时间顺序，看到所有版本的哈希值。

  • 解决冲突

    返回原文件，手动处理

  • 修改上一条提交信息

    用到的时候，再来整理吧。

  • 压缩历史（两次提交可以压缩为一个历史）

4 推送至远程仓库

  • 添加远程仓库

    本次操作过后，git会自动将远程仓库的名称设为origin

  • 推送master分支

    可能会connection timed out，再试一次就好了。

    -u可以在推送的同时，将origin仓库的master分支设置为本地仓库当前分支的upstream，将来 git pull 的时候，本地仓库的这个分支可以直接从 origin 的 master 分支获取内容。

  • 推送master以外的分支
  • 修改本地仓库所对应的远程仓库

5 从远程仓库获取

  • 获取远程仓库
  • -a 参数，查看本地或者是远程分支的信息
  • 获取远程仓库的分支
  • 将修改后的分支推送远程仓库
  • 获取最新的远程仓库分支

6 混淆点

  • 版本历史有两个主线
    时间的顺序。擦掉重写后，分支图的顺序（有一部分没有了）。
  • 退出编辑器

论情商

发表于 2020-03-09 分类于 随笔
本文字数： 624 阅读时长 ≈ 1 分钟

论情商，怎样和这个世界好好相处。

语言会产生误导

大家都说情商高的人会说话。但语言的意义是什么呢？

“你很好！可是我不想理你”，“你可以找到更好的朋友”，“我有什么好的”…这些话，字面上是安慰，其实只是为了达到说话人的目的：我不想理你了，你自己玩吧。

因为人类的语言，很多时候都不是为了诉说真实，大多时候都是为了达到某种目的。领导的话，让一群人向一个目标前进。妈妈对孩子的安慰，让孩子一直乖乖的。情人的话，或许是为了开心。分手的话，都是想摆脱的借口…

所以说，我们真要为达到别人的目的而活吗？当然我们要为实现自己而活。

何为情商高

情商高是得到别人的认可-“你的情商真高”？当然不是，真正的情商高是通过自己的努力，达到了自己的目的。别人的认可是无关紧要，就算别人觉得你情商低都没事。

当然，我现在的情商是低的，不知道别人的目的，达不到自己的目的，还加深自己与这个世界的矛盾。

如何提高情商

知己知彼，百战不殆

想要别人对自己好，得先满足别人的需求，知道别人需要什么，然后才是自己需要什么。而语言只是技巧，不是本质，最本质是语言背后的思维。

承认人的“肤浅”

“肤浅”例如以貌取人，例如拜金…其实也是人类进步的产物，很多时候这种简单粗暴的方法确实让我们活的更好。

这样的大环境我们没有办法改变，可以改变的只有我们自己。让自己变得更有魅力，让别人更愿意去翻阅。

什么样的人情商高

情商高的人，大概就是通过自己的智慧，可以和这个世界的规则好好相处，及不损害他人利益，也不违背自己的初心。
