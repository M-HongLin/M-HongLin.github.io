python基础

发表于 2020-03-16 更新于 2020-04-08 分类于 CS学习笔记 阅读次数：
本文字数： 24k 阅读时长 ≈ 21 分钟

python是一个很容易入门的语言。它简单好用。关键是它可以画图，爬资料。其实我也是冲着这个爬资料来学它的。
离开学校后，最大的不同就是终于不用考试了。很多问题都可以查资料，找别人帮也可以。
记住的都应该是最重要或者是必要的东西，其他的可以翻资料啊

一门好的课程或是一本好书，总是可以用我们乐于接受的方式，帮助我们入门，并且记住一些重要且必要的东西。我用的学习资料是 Eric Matthes编写的《Python编程从入门到实践》。不需要什么基础，非常亲切易读的一本书。本文整理了一些我觉的是重点的东西。

其实学工科的我，一点点编程知识就够用了。这导致我只学了一点点语言知识，然后大言不惭的告诉别人：这个我会呀！这大概也就像小孩子学语言比较快的原因，因为小孩说来说去就那几句话，不需要表达复杂的东西。果然知道的越多，知道自己不知道的也会越多。学习吧…

重要概念

列表 元组 字典 函数 参数传递 返回值 类

1 基础语法

1-1 变量和简单的数据类型

变量

  • 命名有规律
  • 先赋值，再使用。
  • python 可以不指定变量类型。

    字符串

  • 用’或”包起来的都是字符串
  • 用函数修改字符串的大小写
  • 用”+”合并字符串
  • 添加空白:制表符(\t),换行(\n)
  • 删除空白,调用一个函数就可以了。

    数字

  • 整数: 运算什么的和数学里面是一样的。
  • 浮点数: 整数与浮点数运算，默认输出浮点数。
  • str(),将数字转换为字符串，这样就可以完成字符串拼接了。

    1-2 列表(是个很有用的工具)

    列表的定义

  • 可以将任何东西放入列表中，其中元素可以没有任何关系
  • 访问列表元素(索引从0开始)
  • 使用列表中的值

    修改，添加和删除元素

    我们创建的列表大都是动态的，随着程序的运行需要增减里面的元素。
  • 修改元素的值
  • 在列表中添加元素:append(),insert()
  • 从列表中删除元素：del,pop(),remove()

    组织列表

  • 永久性排序:sort()
  • 临时排序: sorted() 有时候用"."调用，有时候直接用。或者什么时候用函数，什么时候用类。
  • 倒着打印列表:reverse(),注意这里不是排序的
  • 确定列表长度: len()

    遍历整个列表: for cat in cats:

    这里的i是个临时变量，占有的内存什么时候回收呢！

    避免缩进错误

    创建数字列表

  • 函数range()
  • 利用range(),list()创建列表
  • 对列表进行简单的统计运算 min(),max(),sum()
  • 列表解析(内置了for循环)

使用列表的一部分(前取后不取，没有全部取，负的最后几个)

  • 切片
  • 遍历切片
  • 复制列表

    元组(tuple)

    列表用来存储一系列可以修改的元素，元组是不可变列表
  • 定义元组
  • 元组的值不可以变，但是元组变量可以重新赋值

1-3 if条件语句

条件测试

  • 检查是否相等：=
  • 检查是否相等时不考录大小写
  • 检查是否不相等：！=
  • 比较数字: <,>,<=,>=,==,!=
  • 检查多个条件: and,or
  • 检查特定值是否在列表中

if 语句

  • if 语句的基本结构

使用 if 语句处理列表

  • 看看列表是不是空的

    1-4 字典

    字典储存的信息量几乎不受限制。字典是动态结构。Python不关心键值对的添加顺序，只关信键值之间的关联。

    定义字典

    字典是一系列键值对，python中任何的对象都可以成为字典中的值。

    使用字典

  • 访问字典中的值
  • 添加键值对
  • 创建空字典
  • 修改字典中的值
  • 删除键值对

    遍历字典

  • 遍历所有的键值对 items()
  • 遍历字典中所有的键 keys(),
    keys()并非只可以用来遍历，实际上它返回一个列表，包含字典中的所有键。
  • 按顺序遍历字典中所有的键:获取字典元素时，获取顺序是不可预测的。要以特定的顺序返回字典元素，可以在for循环中对返回的键进行排序。
  • 遍历字典中所有的值：values()

    嵌套

  • 字典和列表都可以嵌套使用

    1-5 用户输入与循环

    input()工作原理

  • input()获取字符串
  • int()将字符转为数字
  • 求模运算：%

    while 循环简介

  • 让用户选择何时退出
  • 使用break退出循环
  • 使用continue
  • 避免死循环
    Ctrl+C退出循环，或者关闭程序输出的终端窗口

定义函数

  • 函数结构
  • 向函数传递信息

  • 位置实参,关键字实参,默认值

    返回值

  • 返回简单值,字典

    传递列表

  • 不允许函数修改列表

    传递任意数量的实参

1-7 类

创建和使用类

  • 创建dog类
  • 根据dog类创建实例

    使用类和实例

  • 给属性指定默认值，修改属性的值

    继承

    子类继承父类时，可以继承父类所有属性和方法，也可以定义自己的属性和方法。
  • 子类的方法init()
  • 给子类定义属性和方法
  • 重写父类的方法
  • 将实例用作属性
    将类的一部分作为一个独立的类提取出来。可以将类拆成几个协同工作的小类。
  • 模拟实物

    导入类

    python 标准库

    只需用一个import，就可以导入使用标准库里面的所有类与函数

    类编码风格

  • 类名每个单词的首字母大写，且没有_
  • 实例名都小写，用_连接
  • 类中用一个空行分隔方法，模块中两个空行分隔类
  • 程序中，先导入标准库中的，空一行，再导入自己写的

1-8 文件和异常

从文件中读取数据

读取文件时python将所有文本都理解为字符串。

  • 读取整个文件
  • 文件路径
  • 逐行读取
  • 创建一个包含文件各行内容的列表
  • 使用文件的内容
  • 包含一百万位的大型文件
  • 圆周率值中包含你的生日吗

    写入文件

    python只能将字符串写入文件中去。如果想要存入数据，得先将它变成字符串。

open()默认以只读方式打开文件。还可以，’w’:写入；’a’:附加；’r+’:读取和写入

open()的文件不存在，open()将自动创建它。’w’打开存在的文件就要小心了，Python将在返回文件对象前清空该文件。

  • 写入空文件
  • 写入多行
  • 附加到文件

    异常

  • 使用try-except代码块
  • 失败时一声不吭

存储数据

  • 使用json.dump()和json.load()
  • 保存和读取用户生成的数据

    用python模块unittest中的工具来测试代码

    测试函数

  • 最初只需要对代码重要行为进行测试，后面再全覆盖测试。
  • 可通过的测试
    涉及的两个文件
    name.py name_function.py
  • 测试未通过时:修复导致测试不能通过的代码。
  • 添加新测试

测试类

  • 各种断言方法: a == b,a != b,x为True,x为False,item在list中,item不在list中
  • 一个要测试的类
    survey.py language_survey.py
  • 测试AnonymousSurvey类
  • 方法setUp()：unittest.TestCase类中包含方法setUp(),如果包含了setUp,python将先运行它，再运行各种以test开头的方法。这里的self好容易被忘记
